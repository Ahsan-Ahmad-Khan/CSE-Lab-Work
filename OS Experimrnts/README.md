# 🖥️ Operating System Experiments (C Language)

## 📌 Overview
This repository contains all **Operating System Lab experiments** implemented in C language.  
It covers important OS concepts like CPU Scheduling, Memory Management, File Organization, Disk Scheduling, Page Replacement, and Deadlock handling.
